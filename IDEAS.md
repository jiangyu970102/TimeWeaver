# TimeWeaver 功能脑暴清单

> 不设限的功能构想，按板块分类。

---

## 一、时间记录体验优化

| 想法 | 说明 | 复杂度 |
|------|------|--------|
| **时间记录模板** | 预设常用活动组合（如"晨会+编码+Code Review"），一键开始一组计时 | ★☆☆ |
| **批量编辑** | 勾选多条记录后批量修改分类、标签、描述 | ★☆☆ |
| **拖拽调整时间轴** | 在时间轴上直接拖拽修改开始/结束时间 | ★★☆ |
| **日历视图** | 类似 GitHub 贡献图，但按月展示每天的时间投入热力 | ★★☆ |
| **时间线缩放** | 按小时/天/周/月切换显示粒度 | ★★☆ |
| **回溯补录增强** | 日历选择器 + 时间滑块，快速补录过去的时段 | ★★☆ |
| **定时器预设** | 自定义多个计时器（25min 编码 / 15min 会议 / 5min 休息）一键切换 | ★☆☆ |
| **焦点模式** | 开始计时自动进入免打扰，屏蔽非关键通知 | ★★☆ |

## 二、AI 深度集成

| 想法 | 说明 | 复杂度 |
|------|------|--------|
| **AI 自动分类** | 根据记录描述自动推荐最匹配的分类标签 | ★★☆ |
| **AI 日/月总结** | 不仅周报，每日/每月也生成 AI 总结，format 更简短 | ★★☆ |
| **AI 效率评分** | 基于专注时长、任务切换频率、目标完成率等计算每日效率分 | ★☆☆ |
| **AI 目标预测** | 根据当前进度和历史模式预测能否达成今日/本周目标 | ★★☆ |
| **AI 会话内容分析** | 导入 AI 对话内容，分析问题类型（调试/需求/架构）及耗时分布 | ★★★ |
| **AI 降级体验** | 未配置 API Key 时提供每日箴言/数据洞察替代 AI 生成 | ★☆☆ |
| **AI 问答** | 对项目说"我这周哪天的效率最高？"直接自然语言查询 | ★★★ |

## 三、Git 集成增强

| 想法 | 说明 | 复杂度 |
|------|------|--------|
| **文件系统监听** | 使用 WatchService 监听 repo 变化，新提交自动入库，无需手动扫描 | ★★★ |
| **分支维度分析** | 按分支/功能统计编码时间和提交频率 | ★★☆ |
| **commit 类型自动归纳** | 按 feat/fix/docs/refactor 等 Conventional Commits 前缀自动归类 | ★☆☆ |
| **代码变动量化** | 关联 `git diff --stat` 统计每次提交的增删行数 | ★★☆ |
| **跨仓库总览** | 多个仓库的提交在同一条时间线上展示 | ★☆☆ |
| **未提交变更提醒** | 检测到工作区有未提交变更时提醒记录到时间线 | ★★☆ |

## 四、数据可视化

| 想法 | 说明 | 复杂度 |
|------|------|--------|
| **年度时间报告** | 类似 Spotify Wrapped 的年度时间投资报告，可分享 | ★★★ |
| **自定义仪表盘** | 用户拖拽选择展示哪些图表和指标 | ★★★ |
| **桑基图深度** | 展示"分类→项目→具体活动"的完整时间流向 | ★★☆ |
| **生产力天气图** | 每天一个天气图标（晴/多云/雨）代表效率等级，直观 | ★☆☆ |
| **对比分析** | 本周 vs 上周、工作日 vs 周末的详细对比 | ★☆☆ |
| **空窗期分析** | 自动识别一天中的"空白时段"，提示是否漏记 | ★★☆ |

## 五、游戏化与激励

| 想法 | 说明 | 复杂度 |
|------|------|--------|
| **成就系统** | 连续打卡 7 天、首次使用各功能、累计 100 小时等成就徽章 | ★★☆ |
| **等级系统** | 专注时长累积经验值，升级解锁新功能或主题 | ★★☆ |
| **每日挑战** | "今天完成 4 个番茄钟"、"尝试一个新分类"等小任务 | ★★☆ |
| **专注森林集成** | 每完成一个番茄钟种一棵树，累计一片森林 | ★★☆ |
| **连胜 streak** | 每日达成目标连续天数，断掉要重新累积 | ★☆☆ |

## 十、技术架构升级

| 想法 | 说明 | 优先级 |
|------|------|--------|
| **测试覆盖率 > 80%** | 单元测试 + 集成测试 + E2E 测试 | ★★★ |

| **数据导出** | CSV / JSON / Excel 导出，用户数据可移植 | ★☆☆ |
| **操作日志** | 关键操作写入审计日志（谁在何时修改了什么记录） | ★★☆ |

## 优先级建议

### MVP+ 快速落地（★☆☆ 简单但有感）
- 时间记录模板
- 键盘快捷键
- 深色模式完善
- AI 自动分类
- 日历视图
- 年度时间报告

### 差异化亮点（中复杂度，展示工程能力）
- IDE 插件（VS Code）
- 日历同步（Google Calendar）
- 成就系统
- 自定义仪表盘
- 文件系统监听自动 Git 导入

### 长期演进（高复杂度，有真实用户后再做）
- 团队空间
- IM 机器人
- 移动端 App/小程序
- 浏览器扩展
